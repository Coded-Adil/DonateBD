Important FAQ

What is Document Object Model (DOM)?

Document Object Model is a model that is mainly used for Building a website. HTML document can be manipulated by Javascript and elements in the HTML document can be used as an object in JavaScript in DOM

How do you select an element from the DOM?

We can use querySelector to select an element in DOM. If the element has an Id we can also use getElementById to select that element in DOM

What is event delegation on the context of the DOM, and why is it useful?

Event Delegation is a way to use HTML element and make function to let the website work using eventListener. By using event listener we can take an element and add some function to the element

How do you manipulate an element's attributes and style using DOM?

To manipulate an element we have to get the element and set it as a variable in Javascript and then we can use event listener to manipulate that element. To manipulate attributes and styles those elements we can use .addEventListener and then we and create a function and inside that function we can manipulate the element to add attributes and we can also style.
